# Booster

This project now uses [Next.js](https://nextjs.org/) with [TypeScript](https://www.typescriptlang.org/) and [Tailwind CSS](https://tailwindcss.com/) to render the marketing hero component in `src/components/Hero.tsx`.

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

The Tailwind configuration scans files inside `app/` and `src/`, so utility classes from the existing hero component remain available globally through `app/globals.css`.
